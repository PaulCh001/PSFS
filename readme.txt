PSFS - реализация файловой системы в файле.
----

В тестовом задании представлен COM-объект, реализующий файловую систему с минимальным набором функций, содержимое которой сохраняется в файле.
Работа над проектом заняла приблизительно 30 часов чистого рабочего времени.

В API присутствуют следующие методы:

LoadFromFile - загрузить содержимое файловой системы из файла
Flush - сохранить содержимое в файле. Автоматически выполняется при уничтожении объекта
CreateFolder - создать папку
DeleteItem  - удалить папку или файл
CopyItem - скопировать или перенести папку или файл
OpenFile - открыть файл на чтение или запись
CloseFile - закрыть файл
GetFileSize - получить размер файла
SetFilePos - установить позицию чтения/записи
WriteData - записать блок данных
ReadData - прочитать блок данных
PrintOut - (для отладки) вывести содержимое файловой структуры
                                                          

В силу отсутствия жестких рамок относительно функциональности объекта, некоторые присущие реальным объектам функции не были реализованы. Однако 
реализовать их не составляет особой сложности - это вопрос времени.

В частности:
- не реализованы атрибуты файлов и папок (для реализации нужно ввести набор атрибутов и добавить функции в API для манипуляции атрибутами)
- не поддерживается обращение к файлам и папкам по относительному пути (для реализации нужно ввести понятие текущей папки и добавить функции в 
  API для манипуляции этой папкой)
- не поддерживается многозадачность (реализована блокировка открытых файлов, но реализация не является потокобезопасной. Для решения проблемы 
  нужно защитить структуру хранения блокировок через, например, критическую секцию)
- не реализован доступ к файлам по маске (wildcards)
- не реализован механизм autocommit, когда все изменения автоматически сохранялись бы на внешнем носителе
- недостаточно реализованы проверки на корректность входных данных - имен файлов, путей. В реальной системе требуется более тщательный контроль.


PSFS_test - модуль тестирования компонента
---------

Для целей тестирования создан проект PSFS_test, который создаёт/уничтожает объекты PSFS, проверяет основные функции и визуализирует внутреннее 
состояние.
